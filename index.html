    Java-samples

    This repository contains various code snippets written in Java Programming Language.
    It contains a set of folders, each one containing one or more .java files. These snippets feature various topics : math, logic, etc.
    They were implemented for fun, while I was on the process of getting my feet wet with Java Programming.
